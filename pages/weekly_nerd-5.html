KISS (Keep it simple stupid) is een ontwerpprincipe dat benadrukt dat systemen en ontwerpen eenvoudig moeten worden gehouden. Het idee is dat eenvoud vaak leidt tot betere bruikbaarheid, onderhoudbaarheid en betrouwbaarheid.

Principes van KISS:
  • Vermijd onnodige complexiteit.
  • Focus op de kernfunctionaliteit.
  • Maak ontwerpen intuïtief en gemakkelijk te begrijpen.
Voordelen:
  • Vermindert de kans op fouten.
  • Maakt systemen gemakkelijker te onderhouden.
  • Verbetert de gebruikerservaring.
Voorbeelden:
  • Gebruik duidelijke en eenvoudige code in softwareontwikkeling.
  • Ontwerp interfaces met minimale afleiding en een focus op de belangrijkste functies.
  • Vermijd over-engineering in technische oplossingen.

Git bash

Git Bash is een applicatie voor Microsoft Windows die een emulatie van een Unix-shell biedt en de Git-opdrachtregeltools bevat. Het wordt vaak gebruikt door ontwikkelaars om Git-repository's te beheren en met Git te werken via de opdrachtregel.

Belangrijke commando's:
  • git init - Initialiseert een nieuwe Git-repository.
  • git clone [url] - Kopieert een bestaande repository naar je lokale machine.
  • git status - Toont de status van wijzigingen in de repository.
  • git add [bestand] - Voegt wijzigingen toe aan de staging area.
  • git commit -m "bericht" - Slaat wijzigingen op met een commit-bericht.
  • git push - Verzendt lokale commits naar een remote repository.
  • git pull - Haalt wijzigingen op van een remote repository en voegt ze samen.
Voordelen:
  • Ondersteunt krachtige Git-functionaliteiten.
  • Biedt een Unix-achtige omgeving op Windows.
  • Compatibel met shell-scripts en andere Unix-tools.

Repair cafe

Het Repair Café is een bijeenkomst waar vrijwilligers helpen bij het repareren van kapotte spullen. Het doel is om afval te verminderen en mensen te leren hoe ze hun spullen kunnen herstellen. Neem je kapotte apparaten, kleding of andere items mee en werk samen met deskundige vrijwilligers om ze te repareren.

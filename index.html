Datum: 2025/03/24 - 2025/03/27
Onderwerp: Woman in tech

Het idee

Het idee van onze Hackathon was om een wie ben ik spel te maken in de vorm van wie ben ik.
Het werkt via kaartjes in een carrousel. Je krijgt een lijst met een namen van de vrouwen in de tech wereld en dat je aan de hand van vragen moet raden wie het is.

Het Resultaat

De uitkomst van de Hackathon is dat we een werkend prototype hebben gemaakt van het spel.
Na de juryleden hebben beoordeeld, hebben we de hackathon gewonnen met ons idee en 3 kaarten voor CSS day behaald.

Reflectie op eigen niveau

Voor de meesterproef heb ik meerdere leerdoelen moeten aangeven. Aan deze leerdoelen heb ik kunnen werken tijdens de meesterproef om hier in te kunnen verbeteren. Hoe ik aan deze leerdoelen heb kunnen werken, lees je hieronder.

Meerdere manieren bedenken om een iets op te lossen

Tijdens de meesterproef zijn er op meerdere momenten problemen geweest die op meerdere manieren opgelost konden worden. Een van die problemen was het responsive maken van een horizontale slider.

Mijn eerste oplossing was om een overflow: hidden; op de container te plaatsen. Dit was echter niet de beste oplossing omdat dit ook problemen gaf voor andere ontwerp ideeÃ«n. Het probleem is daarna op een andere manier opgelost om de website responsive en uitbreidbaar te maken.

Vaker feedback vragen aan klasgenoten & docenten

Tijdens de meesterproef zijn er momenten voor studenten om feedback te vragen aan docenten. Deze momenten heb ik gebruikt om te vragen wat ik het best kon doen op bepaalde momenten zoals communicatie met de opdrachtgever en meer technische vragen zoals server sided code.

Optijd documenteren om te zorgen dat het up- to date blijft

Om te zorgen dat de documentatie up-to date is gebleven heb ik elke week de documentatie bijgewerkt doormiddel van een Product Biografie. Deze biografie is te vinden in de Repository van de meesterproef.

CSS day

Wat is CSS day

De CSS day is een evenement dat plaatsvindt op 2025/04/11 in Amsterdam. Het is een dag vol met presentaties en workshops over CSS en webontwikkeling. Het is een geweldige kans om meer te leren over CSS en om in contact te komen met andere ontwikkelaars.
